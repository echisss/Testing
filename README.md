**Баг 1.** В конструкторе пиццы кнопка «Тонкое» остается кликабельной при выборе диаметра пиццы 25 см.

**Описание:**

При выборе диаметра пиццы 25 см кнопка «Тонкое» остается кликабельной.

**Ссылка на требования:**

Документация к сайту «Конструктор пиццы Papa German» — https://buildin.ai/qa-studio/share/dc6bbbf1-0d3f-4219-bee3-d64f958434c6?code=2RZ6CK

**Предусловия:**

Открыта страница сайта https://open.qastudio.ru/manual-qa/lessons/mentor-reviewed-practice/papa-german

**Шаги воспроизведения:**

1\. Нажать на кнопку «25 см», выбрав диаметр пиццы

2\. Нажать на кнопку «Тонкое», выбрав вид теста пиццы

**Текущий результат:**

При нажатии на кнопку «Тонкое», ранее нажав на кнопку «25 см», выбирая диаметр пиццы, выбор параметра «Тонкое» остается кликабельным.

**Баг 2.** В конструкторе пиццы некорректно работает кнопка недоступной добавки «Шампиньоны».

**Описание:**

Кнопка с недоступным ингредиентом «Шампиньоны» остается кликабельной. При ее нажатии недоступный ингредиент «Шампиньоны» можно добавить к пицце, а также стоимость недоступного ингредиента прибавляется к общей цене пиццы на кнопке «В корзину за N ₽».

**Ссылка на требования:**

Документация к сайту «Конструктор пиццы Papa German» — https://buildin.ai/qa-studio/share/dc6bbbf1-0d3f-4219-bee3-d64f958434c6?code=2RZ6CK

**Предусловия:**

Открыта страница сайта https://open.qastudio.ru/manual-qa/lessons/mentor-reviewed-practice/papa-german

**Шаги воспроизведения:**

1\. Нажать на кнопку «20 см» при выборе диаметра пиццы

2\. Нажать на кнопку «Традиционное» при выборе вида теста пиццы

3\. Нажать на кнопку с недоступной добавкой к пицце «Шампиньоны»

**Текущий результат:**

Кнопка с недоступным ингредиентом «Шампиньоны» остается кликабельной (при выборе добавки вокруг кнопки появляется оранжевая рамка), при этом при нажатии этой кнопки «Шампиньоны» можно добавить к пицце. Кроме того, стоимость недоступного ингредиента прибавляется к общей цене пиццы на кнопке «В корзину за N ₽», то есть вместо нужных 219 рублей итоговая стоимость равна 398 рублей.

\- Недоступный ингредиент нельзя добавить к пицце

\- Стоимость недоступного ингредиента «Шампиньоны» не может прибавляться к общей цене пиццы на кнопке «В корзину за N ₽»

**Баг 3.** В конструкторе пиццы сырный бортик добавляется для тонкого теста пиццы.

**Описание:**

При выборе сырного бортика в качестве дополнительного ингредиента его можно добавить для пиццы с тонким видом теста.

**Ссылка на требования:**

Документация к сайту «Конструктор пиццы Papa German» — https://buildin.ai/qa-studio/share/dc6bbbf1-0d3f-4219-bee3-d64f958434c6?code=2RZ6CK

**Предусловия:**

Открыта страница сайта https://open.qastudio.ru/manual-qa/lessons/mentor-reviewed-practice/papa-german

**Шаги воспроизведения:**

1\. Нажать на кнопку «35 см», выбрав диаметр пиццы

2\. Нажать на кнопку «Тонкое», выбрав вид теста для пиццы

3\. Нажать на кнопку с доступной добавкой к пицце «Сырный бортик»

**Текущий результат:**

При нажатии кнопки «Сырный бортик» в качестве добавки к пицце диаметром 35 см с тонким тестом появляется возможность применить ее.
